# Changelog

All notable changes to vpype-penset will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-01

### Added

- Pen and PenSet data model for vpype plotter plugins
- Built-in artistic and plotter-oriented pen sets
- Unified `penset` command accepting names, hex colors, or TOML files
- `colorize`, `peninfo`, and `pensets` vpype commands
- TOML-based pen set loading for physical pen inventories
- Tests covering pen set parsing, metadata flow, and command behavior
